# PlayStation Games Performance Dashboard

## Project Overview
This project analyzes PlayStation games using Power BI. The dashboard focuses on game ratings, platforms, publishers, prices, release trends, and player reception.

## Tools Used
- Power BI
- Power Query
- Excel

## Dashboard Features
- Total number of games
- Average PlayStation score
- Average Metacritic score
- Total PlayStation reviews
- Average price
- Top games by PlayStation score
- Rated games by category
- Number of games by platform
- Games released over time
- Top publishers by number of games

## Key Insights
- PS5 has the highest number of games in the dataset.
- Most rated games fall between Good, Very Good, and Average categories.
- Game releases increased strongly in recent years.
- Publisher activity varies across the dataset.
